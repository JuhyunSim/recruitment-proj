# recruitment-proj
- 스프링 부트를 활용한 채용 공고 시스템

## 채용 공고 관련 기능
1. [ v ] 채용 공고 등록
2.  채용 공고 조회  
  - [ v ] 등록된 채용 공고 전체 조회  
  - [ v ] 특정 채용 공고 조회(1개)
3. [ v ] 채용 공고 수정
4. [ v ] 채용 공고 삭제
## 이력서 관련 기능
1. [ ] 이력서 작성 및 저장
2.  이력서 공고 조회
  - [ ] 저장된(작성 중 or 지원완료) 이력서 모두 조회
  - [ ] 특정 이력서 조회(1개)
3. [ ] 이력서 수정
4. [ ] 이력서 삭제
## 지원 시스템
1. [ ] 지원하기
2. [ ] 지원결과 확인
